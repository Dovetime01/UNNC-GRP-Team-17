{{< lead >}}
This section addresses the practical challenges encountered during on-set virtual production work, focusing on managing dynamic scene changes and understanding the lighting framework that supports real-time decision-making.
{{< /lead >}}

## Learning Outcome

By the end of this section, learners should be able to anticipate and adapt to last-minute set modifications, understand UE lighting architecture and tools, and apply lighting frameworks that enable rapid iteration during production.

## This Section Covers

1. Responding to emergent on-set requirements and scene modifications
2. On-set asset generation using AI tools under time pressure
3. Asset combination and iteration workflows when existing resources are available
4. Virtual lighting systems and actors in Unreal Engine
5. Lighting baselines and snapshot-based light management for multi-shot scenes
